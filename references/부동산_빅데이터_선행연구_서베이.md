## 1️⃣ 가격 예측 (머신러닝/딥러닝)

### 조민지 (2023) — ⭐한양대 부동산융합대학원 석사
**"서울시 아파트 매매가격지수 예측력 비교 연구: 시계열과 딥러닝 모형 중심으로"**
- KB 주택매매가격지수 + 한국부동산원 실거래가격지수
- 30개 거시경제변수 다변량 분석
- 시계열(VECM) vs 딥러닝(LSTM, GRU) 비교
- 그랜저 인과관계 → 공적분 검정 → VECM 설계
- 딥러닝이 시계열보다 예측력 높음
- 단, 시계열은 통계적 해석 가능 → 병행 사용 권장
- **💡 같은 학교(한양대) 선행 → 직접 레퍼런스 가능**

### 이학만 (2025) — 부경대 석사
**"머신러닝을 활용한 부산시 부동산 지수 분석 및 예측"**
- LSTM 다변량 모형
- 입력: 부산 지가지수, 전세가격지수, 소비심리지수 → 출력: 아파트매매지수
- 예측정확도 0.98, 오차율 0.06
- 한계: 구 단위 세분화 부재, 변수 제한적

### 진수정 (2024) — 서울대 석사
**"SRGCNN 모형을 이용한 서울시 아파트 매매 가격 예측"**
- Spatial Regression Graph Convolutional Neural Networks
- 지리적 위치 기반 공간적 패턴 포착
- 공간 가중치 행렬 최적화 탐색
- 크리깅(전통 공간예측) 대비 성능 비교
- **💡 공간 데이터 + 딥러닝 결합 = 최신 트렌드**

### 김선현 (2022) — 경북대 석사
**"빅데이터 기반의 아파트 가격 형성 요인 분석 및 가격 예측을 위한 다중회귀모델 연구: 대구광역시"**
- 128개 법정동 분류
- 공시가격 + 실거래가격 → 다중회귀분석
- 결정요인: 전용면적, 경과연수, 층수, 학원 수, 세대수/면적, 1인당 공원면적
- 공원면적 증가 시 가격 하락 예상과 반대 결과

### 이선구 (2024) — 건국대 박사
**"딥러닝 모형을 활용한 부동산 조각투자 가격 예측: 리츠와의 비교"**
- LSTM vs XGBoost vs VAR 모형 비교
- 토큰증권(STO) → LSTM 적합 / 리츠 → XGBoost 적합
- 거시경제변수 추가 시 예측 성능 차이 미미
- **💡 조각투자(STO) = 신규 연구 영역**

## 2️⃣ 텍스트 마이닝 / 감성분석

### 박재수 (2020) — 강원대 석사
**"주택시장 예측을 위한 부동산 감성지수 개발 연구: 뉴스와 방송 빅데이터에 AI 적용"**
- 신문 3개 + 경제지 3개 + 방송 3사 웹크롤링
- 토픽모델링 + TextRank + TF-IDF + 나이브베이즈
- 신문 감성지수(NPSI), 방송 감성지수(TVSI) 개발
- ARIMAX 모형에 감성지수 투입 → 예측력 향상 (RMSE 7.9%, MAE 6.2%)
- 신문 감성지수 > 방송 감성지수 (예측 기여도)
- **💡 감성분석의 대표 선행연구**

### 오성훈 (2022) — 호서대 박사
**"뉴스 빅데이터를 이용한 부동산가격 변화 시점 예측"**
- 가격 수준이 아닌 **추세 전환 시점** 예측
- 주간 아파트 매매가격지수 증가율 활용
- 긍정/부정 단어 빈도 → 대응단어쌍 생성
- (상승세×100)/(상승세+하락세)-50 지수 변환
- **💡 "가격 얼마" 대신 "언제 바뀌나"에 초점 = 차별화**

### 이현정 (2023) — 동의대 박사
**"텍스트 마이닝을 이용한 부동산경기 순환기별 투자심리 특성 연구"**
- 빅카인즈 54개 매체 / 2012~2022 (10년)
- LDA 토픽 모델링
- 3개 순환기(확장/수축) 구분 분석
- 제1순환기: 정책 영향 / 제2순환기: 금리 영향 / 제3순환기: COVID-19
- 밴드왜건효과, 수도권 쏠림 현상 확인
- **💡 순환기별 분석 = 시계열 구조화 방법론**

### 이용운 (2024) — 광운대 석사
**"빅데이터를 활용한 부동산 젠트리피케이션 감성분석 연구"**
- 텍스톰(Textom) + UCINET + NetDraw
- 네이버 웹문서/블로그/카페/뉴스 (2013~2022)
- TF-IDF, 2-gram, 연결중심성, CONCOR 분석
- 감성분석 결과: 긍정 67.22% vs 부정 32.78%
- **💡 소셜미디어 감성분석 기법 참고**

- **💡 사회과학적 접근 (설문 + 구조방정식)**

## 🔍 연구 공백 (Research Gap)

위 서베이를 바탕으로 아직 덜 다뤄진 영역:

1. **LLM/생성AI 기반 부동산 텍스트 분석** — 기존은 TF-IDF, LDA, 나이브베이즈 수준. GPT/BERT 적용 논문 국내 부동산 분야 거의 없음
2. **XAI(설명가능AI) 부동산 적용** — 예측만 하고 "왜"를 설명하는 연구 부족 (SHAP/LIME)
3. **실시간 스트리밍 데이터 분석** — 기존 연구는 모두 배치 분석. 실시간 크롤링+예측 없음
4. **조각투자(STO)/토큰증권** — 박사 1편만 있고 석사 수준 연구 부재
5. **인구이동 × 주택가격** — 코로나 이후 데이터로 분석한 연구 적음
6. **다중모달 분석** — 텍스트+이미지+수치 결합 분석 전무
7. **한양대 자체 후속연구** — 조민지(2023) LSTM/GRU 연구의 확장 가능성

## 🔥 해외 논문 기반 — 국내 미개척 주제 TOP 5

### 1️⃣ XAI + 한국 아파트 가격 결정요인 ⭐⭐⭐ 최강 추천
- **레퍼런스:** Neves(2024), Dou(2023), Kee(2025), Tchuente(2024)
- **방법:** XGBoost/LightGBM → SHAP values로 "왜 이 가격인가" 설명
- **한국 적용:** 국토부 실거래가 공공데이터 + 학군/교통/환경 변수
- **차별화:** 해외는 리스본, 상하이, 이스탄불, 런던 → 서울/수도권 없음
- **난이도:** ★★☆ (데이터 쉬움, 코드 오픈소스 많음)
- **표절 걱정:** 없음 (해외 방법론을 한국 데이터에 적용 = 연구 기여)

### 2️⃣ LLM 기반 한국 부동산 감성지수 개발 ⭐⭐⭐
- **레퍼런스:** Chen & Zhao(2024), Razali(2025), Janowski(2024)
- **방법:** GPT-4/KoBERT로 네이버 부동산카페/뉴스 감성분류 → 감성지수 구축 → 가격 예측력 검증
- **차별화:** 기존 국내 연구는 TF-IDF/나이브베이즈(박재수 2020) → LLM 세대교체
- **난이도:** ★★★ (LLM API 비용, 프롬프트 설계)

### 3️⃣ XAI + 공정성(Fairness) — 부동산 자동감정의 편향 분석 ⭐⭐
- **레퍼런스:** Acharya et al.(2024) IEEE Access 62회 인용
- **방법:** ML 가격예측 모형의 지역/소득 편향을 SHAP으로 진단
- **사회과학적 의의:** 부동산 불평등, 정책 시사점 강함
- **난이도:** ★★★☆

### 4️⃣ 스트리트뷰 이미지 + 주택가격 (멀티모달)
- **레퍼런스:** Dou et al.(2023) — 상하이 street view 활용
- **방법:** 네이버/카카오 로드뷰 이미지 → CNN 특징 추출 → 가격 변수로 투입
- **차별화:** 한국 로드뷰 데이터 활용 연구 전무
- **난이도:** ★★★★

### 5️⃣ 상업용 부동산 ML 가격지수 구축
- **레퍼런스:** Calainho et al.(2024) 48회 인용
- **차별화:** 국내 ML 연구는 아파트 편중 → 오피스/상가 분석 공백
- **난이도:** ★★★☆ (데이터 확보가 관건)

## 📝 서베이 요약

- **가장 많은 연구**: 아파트 가격 예측 (LSTM 중심)
- **떠오르는 분야**: 텍스트 마이닝/감성분석 + 부동산
- **방법론 트렌드**: 시계열 → 딥러닝 → 공간모형(GCN) → (다음: LLM/XAI?)
- **데이터 소스**: KB국민은행, 한국부동산원, 빅카인즈, 텍스톰, 네이버 크롤링
- **한양대 부동산융합대학원**: 조민지(2023) 논문이 직접 선행연구 → 후속연구 설계 유리
