**소속:** 한양대학교 부동산융합대학원 도시부동산정책전공

## 총평

> 본 논문은 서울시 아파트 매매가격 결정요인 분석이라는 학술적·사회적으로 중요한 주제를, 머신러닝(XGBoost)과 설명가능한 인공지능(SHAP)이라는 최신 방법론으로 접근한 수준 높은 석사논문이다. 전체적인 논문 구조는 체계적이며, 분석 코드와 결과 데이터가 논문 본문과 잘 정합되어 있고, 선행연구 검토도 국내외 문헌을 광범위하게 포괄하고 있다. 다만 통과를 위해 반드시 수정해야 할 사항과 강화가 필요한 부분이 존재하며, 이하에서 8개 영역으로 구분하여 상세히 제시한다.

## 2. 내용 충실도

### 2.1 제1장 서론 (전반적으로 우수)

**강점:**
- 연구의 배경이 구체적 통계 수치(부동산 자산 비중 73%)와 함께 제시되어 설득력 있음
- 연구 목적이 4가지로 명확하게 구조화됨
- 논문 구성 설명(제3절)이 각 장의 내용을 정확히 안내함

**보완 필요:**
- 통계청(2024) 인용에서 "가구 평균 자산 중 부동산 73%"는 구체적 출처(통계청 가계금융복지조사 등) 명시 필요
- 연구의 범위(제2절)에서 연구설계서에 기술된 기간이 "2020.01~2025.12"인데, 논문 본문은 "2019.01~2024.12"로 수정됨 — 이 불일치는 수정된 것으로 이해되나, 모든 문서의 기간 표기를 일치시킬 것

### 2.2 제2장 이론적 배경 및 선행연구 (우수)

**강점:**
- Lancaster(1966)와 Rosen(1974)의 헤도닉 가격모형 이론을 수식과 함께 명확히 설명
- XGBoost, 랜덤 포레스트, SHAP의 수학적 정의가 모두 포함됨
- 국내·해외 선행연구의 균형 잡힌 검토

**보완 필요:**
- **Lundberg et al.(2020)의 TreeSHAP 인용 오류:** 본문 270행에서 "Lundberg et al.(2020)"을 TreeSHAP 알고리즘으로 인용하나, 참고문헌 목록에 이 2020년 논문이 없음. Lundberg et al. (2020)의 정확한 서지정보(논문 제목: "From local explanations to global understanding with explainable AI for trees", *Nature Machine Intelligence*)를 추가하거나, 없으면 인용 삭제 필요 (아래 참고문헌 검토 섹션 참조)
- `ALE(Accumulated Local Effects)` 관련 설명에서 "Krämer et al.(2023)"을 ALE의 출처로 언급하나, ALE의 원논문은 Apley & Zhu(2020)임. Krämer et al.(2023)은 ALE를 활용한 응용 논문임. 이론 설명에서 원출처로 수정할 것
- 선행연구 검토에서 Na, Ko & Park(2025)와 Park, Oh & Won(2024)을 간략히 언급하나, 이 두 논문이 본 연구와 가장 유사한데도 차별성 설명이 부족함. 구체적 방법론·데이터 차이를 서술 보강 필요

### 2.3 제3장 연구 설계 (양호)

**강점:**
- 변수 구성의 4대 분류(물리적/입지/환경/거시경제)가 체계적
- `<표 1>` 독립변수 정의 및 데이터 출처 표가 완성도 높음
- 8:2 train-test split과 5-fold CV 방법이 명시됨

**보완 필요:**

1. **시간적 누출(Data Leakage) 문제:** 8:2 단순 랜덤 분할은 시계열 데이터에서 미래 데이터가 학습에 포함될 수 있는 **시간적 누출** 문제를 야기할 수 있음. 2019~2022년 학습, 2023~2024년 테스트로 시계열 순서를 지킨 분할 또는 Rolling Window 방식을 고려하거나, 랜덤 분할을 유지하되 "연도별 층화 추출(stratified sampling)"로 각 연도가 학습·테스트에 균형 있게 분포되도록 처리해야 심사 시 지적받을 가능성이 큼

2. **OLS 회귀 가정 진단 부족:** 제3절에서 "다중공선성(VIF), 잔차의 정규성, 등분산성 등 회귀 가정의 충족 여부를 진단하였다"고 서술하나, 실제 분석 결과(VIF 값, 정규성 검정 결과 등)가 제4장에 제시되지 않음. 최소한 VIF 결과와 Breusch-Pagan 검정 결과를 표로 추가 필요

3. **연구설계서 대비 누락 변수:** 연구설계서(연구설계서_XAI_아파트가격.md)에 포함됐던 다음 변수들이 최종 논문에서 삭제됨. 논문에서 이유를 명시하지 않으면 심사위원의 지적 대상이 됨:
   - 지하철역 거리 (입지 특성에서 빠짐)
   - 총 세대수
   - 버스정류장 수
   - 범죄율
   - 미세먼지 농도
   - 전세가율
   
   이들이 누락된 이유(데이터 수집 실패, 다중공선성, 자치구 단위와 맞지 않는 변수 등)를 제3장에 한 단락으로 명시할 것

4. **GridSearchCV 하이퍼파라미터 결과 미제시:** "GridSearchCV를 통해 최적값을 탐색하였다"고 언급하나, 최적 하이퍼파라미터 조합이 어디에도 제시되어 있지 않음. `<표>`로 최종 하이퍼파라미터 설정값을 제시할 것 (또는 Early Stopping 사용 시 이를 명시)

### 2.4 제4장 실증 분석 (핵심 장, 보완 필요)

**강점:**
- SHAP 분석 결과의 해석이 상세하고 논리적임
- 강남/비강남 비교분석이 구체적인 수치와 함께 제시됨
- OLS와 SHAP 결과 비교 분석(제3절 2항)이 방법론적 기여를 잘 부각시킴

**보완 필요:**

1. **`<표 2>` 기술통계량 미완성:** 최솟값, 중앙값, 최댓값 등이 "-"로 공란 처리됨. `data/apartment_final.csv`를 활용하여 실제 통계값으로 채워야 함

2. **`<표 3>` 상관관계 행렬 미완성:** 본문은 상관관계를 서술하나 실제 행렬 표가 없음. 피어슨 상관계수 행렬을 표로 추가 필요. 특히 거시경제 변수 간 다중공선성 논의를 위해 필수

3. **`<표 4>` OLS 결과표 미완성:** 표준오차, t값, 수정R², F통계량 등이 누락됨. `scripts/modeling.py`에서 `statsmodels` 패키지를 활용한 OLS 결과 산출 필요

4. **SHAP 시각화 결과물 전면 부재:** `<그림 4~9>`에 해당하는 SHAP Summary Plot, Bar Plot, Dependence Plot, Waterfall Plot이 모두 텍스트 설명으로만 대체되어 있음. 이는 논문의 핵심 기여인 XAI 분석 결과를 시각적으로 증명하지 못하는 치명적 약점임. **실제 Python 시각화 결과 이미지를 삽입하는 것이 최우선 과제**

5. **`<표 6>` SHAP 변수 중요도 미완성:** 11~14위 변수(CD금리, 기준금리, 초등학교수, 고등학교수)의 SHAP값이 누락됨. `results/feature_importance.csv`에 실제 값이 있으므로 표를 완성할 것:
   - CD금리: 2,087만원
   - 초등학교수: 3,245만원
   - 고등학교수: 1,353만원
   - 기준금리: 949만원

6. **SHAP Dependence Plot 해석의 근거 부족:** 전용면적의 "소형→중형 구간 급격한 SHAP값 증가", 건물연령의 "U자형 또는 J자형 패턴"을 서술하나, 이는 실제 그림 없이 "~로 예상된다", "~로 나타날 수 있다"는 가정적 표현으로 서술됨. 실제 Dependence Plot 결과를 삽입하고, 관찰된 패턴에 대한 확정적 서술로 수정 필요

7. **강남/비강남 지역별 모형이 전체 모형의 부분집합인지 별도 모형인지 불명확:** 제4절에서 "강남3구와 비강남 지역에 대해 별도의 XGBoost 모형을 구축"했다고 서술하나, `modeling.py` 코드를 보면 전체 모형으로 학습 후 테스트 데이터를 강남/비강남으로 나누어 평가하는 방식임. 이는 "별도 모형"이 아닌 "전체 모형의 하위집합 평가"임. 논문의 표현과 실제 분석이 불일치하므로 수정 필요

## 3. 방법론 검증

### 3.1 OLS → RF → XGBoost → SHAP 분석 흐름의 학술적 타당성

전체적으로 **타당한 분석 흐름**이며 선행연구(Kee & Ho, 2025; Tchuente, 2024)와 일관됨. 세부 검토 결과:

| 단계 | 방법 | 타당성 | 비고 |
|------|------|--------|------|
| 베이스라인 | OLS (LinearRegression) | ✅ | 표준화 전처리 후 OLS 사용 — 해석에 주의 필요 (아래 참조) |
| 비교모형 | Random Forest | ✅ | n_estimators=200, max_depth=15 |
| 주모형 | XGBoost | ✅ | n_estimators=500, max_depth=8, lr=0.05 |
| 해석 | TreeSHAP | ✅ | shap.TreeExplainer 사용 |
| 평가지표 | R², RMSE, MAE, MAPE | ✅ | 4개 지표 사용, 적절 |
| 데이터 분할 | 8:2 random split | ⚠️ | 시계열 데이터 누출 가능성 |
| 하이퍼파라미터 | 5-fold CV 언급 | ⚠️ | 코드에서 GridSearchCV 미사용 (early_stopping 사용) |

**[주요 방법론적 이슈]**

**이슈 1: OLS 표준화(Standardization)와 해귀계수 해석**  
`modeling.py`에서 OLS 분석 시 `StandardScaler`로 표준화된 X를 사용하나, `<표 4>` 및 본문에서는 "전용면적 1㎡ 증가 시 매매가격이 평균 약 3억 8천만 원 상승"이라는 비표준화 계수로 해석하고 있음. `results/feature_importance.csv`의 OLS 계수(37,991)가 실제 비표준화 계수인지, 아니면 표준화 데이터에서의 계수인지 불분명. `statsmodels`의 OLS(`sm.OLS`)를 비표준화 데이터로 재실행하여 정확한 비표준화 계수와 t값, p값, 95% 신뢰구간을 제시할 것을 권고

**이슈 2: 5-Fold CV와 Early Stopping의 혼용**  
논문에서는 "5-Fold CV 기반 GridSearchCV"를 사용했다고 서술하나, 실제 코드(`modeling.py` line 198)에서 XGBoost는 `early_stopping_rounds=50`으로 early stopping을 사용하고 GridSearchCV는 없음. RF도 GridSearchCV 없이 고정 하이퍼파라미터를 사용. 논문에서 "GridSearchCV를 통해 최적값을 탐색하였다"는 서술이 실제 분석과 불일치. 실제로 어떤 방법을 사용했는지(early stopping 또는 기본값 설정)를 정확히 서술하거나, 실제로 GridSearchCV를 실행하여 최적 하이퍼파라미터를 도출할 것

**이슈 3: 환경 변수의 자치구 단위 집계 문제**  
공원수, 백화점수, 학교수가 자치구(區) 단위로 집계되어 있어, 같은 자치구 내 모든 아파트에 동일한 환경 값이 부여됨. 이는 자치구 내 입지 다양성을 완전히 무시하는 한계로, 제5장 한계 절에서 이미 인정하고 있으나, 제3장 변수 설정 절에서도 이를 명시하고 왜 자치구 단위를 택했는지(동 단위 데이터 부재 등) 정당화할 것

**이슈 4: 다중공선성 미처리**  
기준금리와 CD금리 간 상관관계는 매우 높을 것으로 예상됨(실제 부동산 시장에서 두 변수는 0.95 이상 상관). 또한 총학교수 = 초등학교수 + 중학교수 + 고등학교수 관계가 성립하여 완전 다중공선성에 가까운 문제 발생. VIF 분석 결과를 제시하고, 다중공선성이 심각한 경우 변수 제거 또는 PCA 처리 방안을 논의할 것

### 3.2 SHAP 분석의 학술적 적절성

SHAP TreeExplainer를 XGBoost에 적용한 방법은 **학술적으로 완전히 타당**하며 Lundberg & Lee(2017)의 원논문 방법론과 일치함. 다만:

- 테스트 데이터 중 5,000건 샘플링은 적절하나, 이 사실이 논문 본문에 명시되지 않음. "전체 테스트 데이터(61,705건) 중 무작위 5,000건을 추출하여 SHAP 분석을 수행하였다"는 문장 추가 필요
- SHAP base value(기준값)가 본문에 제시되지 않음. Waterfall Plot 해석에서 "기본값"이 구체적으로 얼마인지 서술 필요

**연결 누락 또는 약한 부분:**

1. **서베이에 포함된 Rico-Juan & Taltavull de La Paz(2021):** 서베이에서 중요하게 언급("인용: ~95")하나, 논문 본문 선행연구에서는 단 한 줄만 언급됨(제4절). ML+XAI vs 공간 헤도닉 모형 비교라는 방법론적 관련성을 고려하면 더 상세한 논의 필요

3. **Han et al.(2025) PLOS ONE:** 서베이 S4에 포함되어 있으나 논문 미인용. "해석가능 ML + 비선형 효과"를 다루는 최신 연구로 인용 가치 있음

4. **공간 이질성 이론적 기반(Anselin 1988, Fotheringham et al. 2002):** 한계 절에서 인용되나, 제2절 이론적 배경에 공간 이질성 개념을 간략히 소개하는 소절을 추가하면 강남/비강남 비교분석의 이론적 근거가 강화됨

### 5.2 선행연구 검토의 구조적 제안

현재 선행연구 검토는 "국내 → 해외 → 차별성" 순으로 구성되어 있어 양호하나, **방법론별로 재구성**하면 더 체계적:
- 헤도닉 모형 연구 → OLS 기반 선행연구 검토
- 머신러닝 예측 연구 → RF, XGBoost, LSTM 비교
- XAI 기반 해석 연구 → 직접 연관 선행연구 집중 검토

1. "이질적 재화인 부동산의 가격이 면적, 층수, 건축연도, 입지 등 다양한 속성들의 함수로 결정된다고 설명한다." → 주어-서술어 연결이 어색. "...속성들의 함수로 결정된다는 것을 설명한다." 또는 "...속성들의 함수로 결정됨을 설명한다."

2. "블랙박스(Black Box) 특성으로 인해 가격 형성 메커니즘을 해석하기 어렵다는 문제가 존재한다." → "...해석하기 어렵다는 한계가 있다."

3. "이론적으로 일관성 있게 분해할 수 있는 프레임워크이다." → "이론적으로 일관성 있는 분해를 보장하는 프레임워크이다."

4. "이러한 결과는 주택 정책 수립 시 지역별 차별화된 접근이 필요함을 시사하며, Dou et al.(2023)이 상하이 시장에서 확인한 근린환경의 미시적 가격 영향과도 맥을 같이 한다." → "맥을 같이 한다" → "일맥상통한다" 또는 "부합하는 결과이다"

## 8. 개선 제안

### 8.1 논문 통과를 위해 반드시 수정해야 할 사항 (Critical)

> 아래 사항이 미완성 상태로 제출될 경우 심사 통과가 어려울 수 있음

| 우선순위 | 수정 사항 | 이유 |
|---------|-----------|------|
| 🔴 1 | **SHAP 시각화 이미지 삽입** (그림 4~9 전체) | 논문의 핵심 기여 증거 부재 |
| 🔴 2 | **기술통계량 표 완성** (표 2) — 정확한 수치 | 빈칸 제출 불가 |
| 🔴 3 | **상관관계 행렬 표 추가** (표 3) | 다중공선성 논의의 근거 |
| 🔴 4 | **OLS 회귀분석 결과표 완성** (표 4) — t값, 표준오차, VIF | 베이스라인 모형 결과 미완성 |
| 🔴 5 | **참고문헌 누락 추가** (통계청 2024, Lundberg et al. 2020) | 본문 인용-참고문헌 불일치 |
| 🔴 6 | **총 데이터 건수 통일** (308,525 또는 308,555) | 내부 일관성 오류 |
| 🔴 7 | **"별도 XGBoost 모형 구축"→"전체 모형 하위집합 평가"로 표현 수정** | 코드와 논문 서술 불일치 |
| 🔴 8 | **감사의 글 추가** | 학위논문 필수 형식 |

### 8.2 학술적 완성도 향상을 위해 강력 권장하는 사항 (Recommended)

| 권장순위 | 수정/추가 사항 |
|---------|--------------|
| 🟡 A | **시계열 순서를 지킨 데이터 분할** (또는 연도별 층화 추출) 방식으로 변경하거나, 현행 랜덤 분할의 정당성을 논문에서 설명 |
| 🟡 B | **VIF 분석 및 다중공선성 진단 결과** 표 추가 (기준금리-CD금리 간 높은 상관성 논의 필수) |
| 🟡 C | **OLS 분석을 statsmodels로 재실행**, 비표준화 계수·t값·p값·95% CI·VIF·F검정 완전 보고 |
| 🟡 D | **실제 하이퍼파라미터 최적화 과정 기술** (GridSearchCV 실행 또는 early stopping 사용 사실 명시, 최종 하이퍼파라미터 표 제시) |
| 🟡 E | **SHAP base value 명시** — 전체 평균 예측값(기준값) 서술 |
| 🟡 F | **SHAP 5,000건 샘플링 사실 명시** |
| 🟡 G | **Lundberg et al.(2020) TreeSHAP 논문 참고문헌 추가** |
| 🟡 H | **SHAP 변수 중요도 표(표 6) 하위 4개 변수 수치 완성** |

### 8.3 추가하면 논문 수준을 크게 높일 수 있는 사항 (Enhancement)

| 추가 사항 | 기대 효과 |
|-----------|----------|
| **Partial Dependence Plot(PDP) 또는 ALE plot 추가** | 비선형 효과를 SHAP Dependence Plot과 비교하여 방법론적 다각화 |
| **SHAP Interaction Values 분석** | 전용면적×강남구분 상호작용 효과를 정량화하여 강남/비강남 비교분석 심화 |
| **모형별 시기(연도)별 성능 비교** | COVID-19 전후, 금리 인상기 등 시장 국면별 모형 안정성 논의 |
| **공간 시각화 (구별 평균 SHAP값 지도)** | 서울 25개 구별 가격 결정요인의 공간적 분포를 지도로 시각화 |
| **LIME과의 결과 비교** | SHAP 결과의 강건성(robustness) 확인 — 심사위원 질문 대응 강화 |
| **부록: Python 코드 핵심 부분 첨부** | 재현가능성(reproducibility) 보장, 최근 학술지 요구사항 반영 |
