# 📋 논문용 데이터 API 신청 목록

## 연구 변수별 필요 데이터 & 수집처 전체 정리

---

## 🔴 필수 (반드시 신청)

### 1. 국토교통부_실거래가 정보 ⭐핵심
- **용도:** 종속변수 (아파트 매매가격) + 물리적 특성 변수
- **제공 컬럼:** 시군구, 법정동, 아파트명, 전용면적, 층, 건축년도, 거래금액, 계약년월일
- **신청처:** 공공데이터포털 (data.go.kr)
- **URL:** https://www.data.go.kr/dataset/3050988/openapi.do
- **신청방법:** 로그인 → "국토교통부_실거래가 정보" 검색 → 활용신청
- **대안:** API 없이도 https://rt.molit.go.kr → 자료제공 → 엑셀 다운로드 가능

### 2. 서울열린데이터광장 (인증키)
- **용도:** 지하철역 위치, 버스정류장, 공원 등 서울시 데이터
- **신청처:** https://data.seoul.go.kr
- **신청방법:** 회원가입 → 마이페이지 → 인증키 발급
- **이 하나의 키로 아래 데이터 전부 사용 가능:**

| 데이터셋 | 연구 변수 | 비고 |
|---|---|---|
| 서울시 역사마스터 정보 | 지하철역 좌표 (위도/경도) | 거리 계산용 |
| 서울시 버스정류소 위치정보 | 버스정류장 좌표 | 반경 500m 내 개수 |
| 서울시 공원 정보 | 공원 면적/위치 | 반경 1km 공원면적 |
| 서울시 CCTV 설치현황 | 안전 변수 (선택) | 범죄 대리변수 |

### 3. 한국은행 경제통계시스템 (ECOS API)
- **용도:** 거시경제 변수 (기준금리, 소비자물가지수 등)
- **신청처:** https://ecos.bok.or.kr/api/
- **신청방법:** 회원가입 → API 인증키 신청
- **제공 데이터:**

| 데이터 | 연구 변수 |
|---|---|
| 기준금리 | 월별 기준금리 |
| 소비자물가지수 | 물가 통제변수 |
| 통화량 (M2) | 유동성 변수 (선택) |

## 🟡 권장 (있으면 좋음)

### 4. 교육부 학원정보 (나이스)
- **용도:** 학원 수 (학군 변수)
- **신청처:** 공공데이터포털
- **검색어:** "학원 교습소 정보" 또는 "교육부_학원정보"
- **URL:** https://www.data.go.kr 에서 "학원" 검색
- **제공:** 학원명, 주소, 교습과정, 위치 좌표

### 5. 학구도 안내 서비스 (학교 위치)
- **용도:** 초등학교 거리 (학군 변수)
- **신청처:** https://schoolzone.emac.kr 또는 공공데이터포털
- **검색어:** "전국 초중등학교 위치" 또는 "학교정보"
- **제공:** 학교명, 주소, 위도/경도

### 6. 한국부동산원 통계 API
- **용도:** 전세가율, 매매가격지수, 거래량
- **신청처:** https://www.reb.or.kr → 부동산통계정보 → R-ONE
- **대안:** 엑셀 다운로드 가능 (API 없이도 됨)
- **제공 데이터:**

| 데이터 | 연구 변수 |
|---|---|
| 전세가율 | 전세/매매 비율 |
| 아파트 거래량 | 월별 거래량 |
| 소비심리지수 | 시장 심리 |

## 🟢 선택 (없어도 논문 가능, 있으면 풍부해짐)

### 7. 에어코리아 (대기오염 정보)
- **용도:** 미세먼지 농도 (환경 변수)
- **신청처:** https://www.airkorea.or.kr → 공공데이터포털 연동
- **검색어:** "한국환경공단_에어코리아_대기오염정보"

### 8. 경찰청 범죄통계
- **용도:** 자치구별 범죄율
- **수집처:** https://www.police.go.kr → 범죄통계 → 엑셀 다운로드
- **API 불필요** — 연도별 PDF/엑셀 직접 다운로드

### 9. 소상공인시장진흥공단 (상권정보)
- **용도:** 대형마트/백화점 거리, 상권 밀도
- **신청처:** 공공데이터포털
- **검색어:** "소상공인시장진흥공단_상가(상권)정보"

## ⚡ 최소한으로 하려면?

**3개만 신청하면 논문 쓸 수 있음:**
1. ✅ data.go.kr 가입 → 실거래가 + 학원정보 + 학교위치 (한 사이트에서 3개)
2. ✅ data.seoul.go.kr 가입 → 지하철/버스/공원
3. ✅ ecos.bok.or.kr 가입 → 금리/물가

**사이트 3개 가입 = API키 3개 = 데이터 전부 확보**
